Under Windows, you should choose one of these IDE :
- Visual Studio Express 2010
- Dec-C++
- Code::Blocks

Under Visual Studio:
Open the Solution GL4Dummies.sln, compile the Solution (F7) then execute (ctrl-F5)

Under Dev-C++:
- Open the project GL4Dummies.dev then compile the libs (do not execute)
- Open one of the given samples (gl4dWalkOnPolygons.dev, gl4dLandscape.dev, gl4dObjLoader.dev), compile then execute.

Under Code::Blocks:
- Open the project GL4Dummies.cbp then compile the libs (do not execute)
- Open one of the given samples (gl4dWalkOnPolygons.cbp, gl4dLandscape.cbp, gl4dObjLoader.cbp), compile then execute.
